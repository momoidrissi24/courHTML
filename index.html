Les différents type de balises

Il y a 2 type de balises :

  • Les balises usuelles qui encadrent du code html. Le code encadré fini par </balise > ;

  • Les balises opheline qui n'encadrent aucun code et qui se ferment elles-memes ; par exemple :
    <br/ >
